SIGO es una aplicación móvil diseñada para la gestión de usuarios mediante autenticación por matrícula y contraseña, permitiendo a los estudiantes consultar su información personal y su historial académico de forma rápida, segura y desde cualquier lugar.

Funcionalidades Principales

 Inicio de sesión (Login)
Acceso mediante:

Matrícula

Contraseña

 Pantalla Principal
Vista general del sistema después del login exitoso.

 Perfil del Usuario
Muestra la información personal del usuario autenticado:

Matrícula

Datos generales

 Historial Académico
Consulta de calificaciones e información académica registrada.
